MCServer Core Plugin
====================

The Core plugin for MCServer provides the default utility commands and also a lot of WebAdmin goodness.

Commands
--------

 * /back
 * /ban
 * /downfall
 * /give
 * /gm
 * /groups
 * /help
 * /i
 * /item
 * /kick
 * /locate
 * /me
 * /motd
 * /plugins
 * /portal
 * /rank
 * /regen
 * /reload
 * /save-all
 * /spawn
 * /stop
 * /time
 * /top
 * /tp
 * /tpa
 * /tpaccept
 * /unban
 * /viewdistance
 * /worlds

**Also, console commands:**

 * ban
 * banlist
 * getversion
 * help
 * list
 * listgroups
 * numchunks
 * players
 * rank
 * reload
 * say
 * setversion
 * unban  
 * unload
